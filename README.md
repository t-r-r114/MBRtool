## 📖 简介 (Introduction)

MBRtool 是一款基于 Qt/QML 构建的高级磁盘操作与分析工具。它旨在为系统安全研究人员、底层开发者及高级用户提供一套便捷的图形化界面，以实现主引导记录（MBR）的备份、写入、磁盘底层数据编辑（十六进制）及代码反汇编分析。

---

## ✨ 主要功能 (Main Features)

- **MBR 备份与恢复 (`mbrsave` & `mbrwrite`)**：轻松提取或覆盖磁盘的主引导记录扇区。
- **底层物理磁盘读写 (`ntWiter`)**：基于内核级驱动的直接物理磁盘读写能力（支持自定义物理驱动器号及扇区偏移量）。
- **内置十六进制编辑器 (`HexEditor`)**：支持直观地查看、修改文件，并可直接指定物理磁盘（`setTargetDisk`）进行二进制级别的编辑。
- **反汇编分析 (`Disassembler`)**：内置独立的跨窗口反汇编引擎引擎，可直接对引导代码或二进制文件进行机器码级分析。
- **现代化图形界面**：基于 Qt Quick (QML) 打造的流畅、模块化界面，前后端数据实时绑定。

---

## 🛠 技术架构 (Architecture)

- **前端 UI**：Qt QML (采用 Fusion 风格，通过 `setContextProperty` 实现前后端解耦交互)
- **后端逻辑**：C++11/C++17 (结合 Qt 的信号槽机制，包含 `FileReader`、`HexEditor` 等功能模块)
- **系统底层通信**：Windows 系统 API (`DeviceIoControl`, `HANDLE`) 
- **驱动交互模块**：利用 `IOCTL_DISK_WRITE_COMMAND` 直接下发指令至内核层磁盘设备堆栈。
- **构建系统**：CMake 跨平台构建系统。

## ⚠️ 兼容性提示 (Compatibility & Requirements)

### **v2.0.2026.4.18 特别说明：内核隔离冲突**
在 `v2.0.2026.4.18` 及以上的版本中，为实现对物理磁盘的越权底层读写，工具底层引入了 **KDU (Kernel Driver Utility)** 机制来加载或映射相应的驱动程序。
- **不兼容环境**：开启了 **“内核隔离” (Core Isolation / HVCI / VBS)** 的 Windows 系统。KDU 机制与 Windows 的内存完整性安全机制会产生直接冲突。
- **解决方案**：若需正常使用底层物理磁盘写入 (`ntWiter`) 功能，必须进入 Windows 安全中心 -> 设备安全性 -> 内核隔离详细信息，**关闭“内存完整性”**，并重启计算机后再运行本工具。

## 📅 更新日志 (Changelog)

### [v2.0.2026.4.18] - 物理驱动层突破与架构升级
**本次更新完成了从“文件级处理工具”向“内核级驱动控制编辑工具”的蜕变。**
- **新增**：引入 `ntWiter` 核心模块，支持对物理磁盘的直接内核级写入操作（集成 `executeSafeWrite` 及驱动文件的自动下载、解压等环境部署功能）。
- **新增**：`Public.h` 中规范定义了内核通信控制码 `IOCTL_DISK_WRITE_COMMAND` 及 `DISK_WRITE_PARAMS` 结构。
- **增强**：十六进制编辑器 (`HexEditor`) 增加 `setTargetDisk(int diskNumber)` 方法，全面支持针对物理驱动器的直接指向编辑。
- **优化**：大幅提升 QML 交互能力，在全局注册了 `DisassemblyWindows` 类别，并将 `ntWriterBackend` 成功暴露给前端调用。

### [v1.0.0 Formal] - 正式版发布与模块化重构
**标志着 MBRtool 核心功能集的完善与现代 UI 架构的定型。**
- **重构**：全面采用 QML 重写前端界面，实现与 C++ 后端业务逻辑的深度分离。
- **新增**：首次引入 `HexEditor`（十六进制视图）模块，支持稳定加载与保存文件 Hex 数据。
- **新增**：独立的跨窗口反汇编分析模块 (`DisassemblyWindows`)。
- **稳定**：固化 `MBRsave` 与 `MBRwrite` 基础类的读写逻辑。
- **集成**：引入 `FileReader` 供 QML 端直接读取与展示文件内容。

### [v0.1.0 Beta] - 初始测试版
- 核心算法：完成底层 MBR 读取、保存的算法验证。
- 基础模块：搭建了最初的 `MBRsave` 与 `MBRwrite` 的 C++ 框架。
- 工程构建：完成 CMakeLists.txt 的基础配置与编译环境调试。
